#STM-Csharp-tutorial



This project is a tutorial how to configure visual studio to programing on STM32F4 in c# lenguage.

#Overview

Instruction is written in Polish. Here are also implemented examples such as: GPIO, PWM, SPI, RTC, timer.

#Tools, How to run 

In instruction.
